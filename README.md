### 使用说明
******
1. 下载并安装apk
2. 手机对下载的应用开启无障碍模式 MIUI10设置步骤为 设置->更多设置->无障碍->已下载的服务->对main开启服务
3. 打开淘宝喵铺主页
4. 打开安装好的应用即可
